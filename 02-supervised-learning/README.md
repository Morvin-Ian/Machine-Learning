# Supervised Learning

Machine learning algorithms that learn from labeled training data.

## Topics Covered

- **Linear Regression** - Predicting continuous values
  - Simple and multiple linear regression
  - Model evaluation and interpretation

- **Logistic Regression** - Binary and multi-class classification
  - Probability-based classification
  - Decision boundaries and thresholds

- **Classification** - General classification techniques
  - Multi-class classification problems
  - Classification metrics and evaluation

## Learning Path

1. Start with **Linear Regression** to understand regression fundamentals
2. Progress to **Logistic Regression** for classification basics
3. Advance to general **Classification** techniques for complex problems
